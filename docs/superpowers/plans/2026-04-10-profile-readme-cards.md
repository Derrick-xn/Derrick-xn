# Profile README Cards Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restore the broken GitHub profile README cards without paid hosting by generating SVG assets in-repo via GitHub Actions.

**Architecture:** Replace the current public `github-readme-stats.vercel.app` image URLs with local `profile/*.svg` assets. Add a GitHub Actions workflow that regenerates those SVG files on push, on a daily schedule, and on manual dispatch, using an optional PAT secret when present and otherwise falling back to `GITHUB_TOKEN`.

**Tech Stack:** Markdown, GitHub Actions workflow YAML, static SVG assets
